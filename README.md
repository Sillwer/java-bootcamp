# Введение в JAVA
## Обзор
Серия учебных проектов, охватывающих основные аспекты разработки на Java - от базовых конструкций до enterprise-решений.

## Основные направления

### Базовые навыки
- Работа с массивами и структурами данных
- ООП, коллекции, исключения
- Ввод-вывод, работа с файлами

### Многопоточность и сети
- Потоки (Threads), синхронизация
- Сокеты, клиент-серверные приложения
- Паттерны многопоточного программирования

### Базы данных и ORM
- JDBC, работа с PostgreSQL
- JPA/Hibernate, ORM mapping
- Транзакции, connection pooling

### Тестирование и фреймворки
- JUnit, Mockito
- Spring Framework (IoC, DI)
- Конфигурация через XML и аннотации

### Инструменты разработки
- Maven, сборка проектов
- JAR, работа с внешними библиотеками
- Аннотации, кодогенерация

## Ключевые технологии
- Java 8
- Maven
- PostgreSQL, HSQLDB
- Spring Framework
- JUnit 5, Mockito
- JDBC, JPA

Проекты демонстрируют постепенное освоение Java экосистемы - от фундаментальных концепций до построения полноценных приложений с использованием современных фреймворков и инструментов.
